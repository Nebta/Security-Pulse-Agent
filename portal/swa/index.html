Security Pulse Portal scaffold

This is the self-service portal placeholder for the Security Pulse Agent. The build is deferred — see portal/README.md in the repo for the target shape.

When implemented, signed-in customer admins will be able to:

  • Toggle report sections on/off
  • Edit recipient lists and branding
  • Trigger ad-hoc runs
  • View past run history and source-health

Need the report sooner? Contact your security operations team.
